---
title: 'Quickstart'
description: 'Welcome! Follow the steps below to get started with Subtrace.'
---

1. Go to [subtrace.dev/dashboard](https://subtrace.dev/dashboard) and log in with your single sign-on (SSO) identity provider.

2. Click **Get API token** on the top right corner to generate a Subtrace API token.

3. Start a worker node with Docker (remember to set `SUBTRACE_TOKEN` to the token you generated):

```bash
docker run -it --rm \
  -e SUBTRACE_TOKEN=subt_...12345 \
  -v subtrace:/var/lib/clickhouse \
  subtrace.dev/worker:latest
```

4. Install Subtrace using the following command:

```bash
curl -fsSL https://subtrace.dev/install.sh | sh
```

5. Start your backend under Subtrace:

```bash
subtrace run -- python -m http.server
```

Congratulations! Your app is now running under Subtrace. Send a test HTTP
request to your backend and see it appear in the Subtrace dashboard.
